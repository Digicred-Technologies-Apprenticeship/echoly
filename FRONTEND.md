# Echoly — Frontend Pages

1. **Login / Register Page** — User signs up or logs in.
2. **Dashboard** — Overview of GEO health: mention frequency, sentiment, share of voice.
3. **Company Profile Page** — Add and edit company details (name, description, category).
4. **Competitors Page** — Add and view tracked competitors.
5. **Prompts Page** — Add, refine, and run prompts across AI engines.
6. **Results / Echoes Page** — View echoes: engine, mention status, sentiment, and the AI's actual response.
